## 🧠 What You Can Do

This software breaks down complex machine learning tasks into simple steps. You learn how the system processes information through several stages:

### Preparing Data
The tool takes raw text files and converts them into a format the model understands. This process, called tokenization, turns words into numbers.

### Pretraining
This is the phase where the model learns the structures of language. You provide a large amount of text, and the system finds patterns. It learns how words follow one another in sentences.

### SFT (Supervised Fine-Tuning)
After the model learns general language, you guide it to follow specific instructions. You provide question-and-answer pairs so the model learns how to respond to tasks.

### Alignment
The final stage ensures the model acts as you expect. Through feedback loops, such as DPO or GRPO, the model learns to prioritize helpful and accurate responses over irrelevant output.

## ❓ Troubleshooting Common Issues

If you encounter problems, check these solutions:

- "Program does not open": Ensure your graphics drivers are updated to the latest version.
- "Low memory error": Close other applications like web browsers before you start the training process.
- "Slow performance": Check that your computer uses the dedicated graphics card rather than the integrated processor graphics. You can find this setting in the Windows display control panel.
